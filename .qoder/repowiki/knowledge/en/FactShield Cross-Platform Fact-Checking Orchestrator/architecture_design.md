- **Shared Pipeline Logic**: Both the iOS app (`FactCheckCoordinator.swift`) and Chrome extension (`pipeline.js`) implement an identical three-stage verification workflow: Claim Extraction → Evidence Retrieval → Verdict Synthesis, ensuring consistent factual outcomes regardless of entry point.
- **Zero-Dependency Networking**: The iOS module enforces a strict zero-dependency policy, using a custom `actor`-based `APIClient` with `URLSession` for all external communication, while the Chrome extension relies on native `fetch` and `chrome.storage`, eliminating third-party supply chain risks across both platforms.
- **Native System Integration**: The root architecture leverages platform-specific system hooks—iOS `ActivityKit` for Dynamic Island live updates and Chrome `Manifest V3` service workers for background processing—to provide real-time feedback without blocking the user's primary content consumption.
